== Description ==
NexaBlocks is Gutenberg Blocks Library extends the Gutenberg functionality with several unique and feature-rich blocks that help build websites faster. You don't need to have any coding knowledge to build eye-catching websites.  It's for a better page-building experience without the need for traditional page builders like Elementor and Divi.

== Frequently Asked Questions ==


= Does NexaBlocks requires Gutenberg plugin to be installed? =

It does not required Gutenberg plugin. WordPress version 5.9 or above already have Gutenberg integrated so you can use NexaBlocks without installing Gutenberg.

= Why do you need to use a page builder along with Gutenberg? =
A page builder like NexaBlocks offer more rich features and flexibility, which you can't enjoy only using Gutenberg.

= Is NexaBlocks responsive for all screens? =
Yes, it is fully responsive for all devices and tested for cross-browser compatibility.

= Who can benefit from NexaBlocks? =
WordPress users who like to use Gutenberg will love all the advanced and cool blocks of the plugin.

= Does NexaBlocks compatible with other Gutenberg plugins? =

NexaBlocks should be compatible with other Gutenberg plugins, some popular plugins we have checked already.
